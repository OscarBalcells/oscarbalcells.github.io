RedMedic

El PODER de tus datos médicos

Los datos son el oro del siglo XXI. Desde la mejora y personalización de tratamientos, la investigación y creación de nuevos fármacos hasta el ahorro de cientos de miles de millones de dolares al año en sobrecostes en el sistema de salud. La mejora del control y uso de tus datos médicos pueden aportar extraordinarios beneficios tanto a tí mismo, el paciente, como a todo el ecosistema. A día de hoy esta información está en múltiples silos de datos desconectados y es díficil recolectarla, organizarla y compartirla para beneficio de todos.

El objetivo de RedMedic es romper el paradigma actual y aportar un nuevo enfoque al proceso de intercambio de datos de salud. RedMedic aplica la tecnología Blockchain, inhackeable y descentralizado, para crear un sistema de permisos de acceso, regulados por el dueño de los datos, el paciente, mediante una firma digital. Los datos médicos residen en las bases de datos de las instituciones donde se han generado y son accesibles a través de una API abierta y segura que verifica las peticiones utilizando el Blockchain.

Ventajas

Accesibilidad

Una única plataforma que agrega los datos de los diferentes proveedores de salud que visitas. Te ofrece una historia médica longitudinal más fácil de visualizar y compartir con datos filtrados por fecha y proveedor y ordenados por categorías.

Interoperabilidad

Redmedic es una capa que opera por encima de los diferentes silos de datos que poseen los centros de salud. Permite que los diferentes sistemas hablen entre si ya que utiliza un único formato y posee una REST API abierta con autentificación para cada proveedor.

Seguridad

La descentralización de los datos provoca que haya muchos más posibles vectores de ataque con menos daño potencial en vez de un único punto de apoyo para todo el sistema. La Autentificación de las firmas funciona a través de la segura criptografía de curva elíptica (ECDSA).

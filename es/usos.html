RedMedic:   utilidad

¿Para que sirve un sistema de gestión de datos médicos multiinstitucional?

Buena pregunta.

RedMedic es un sistema de gestión de datos médicos. Este sistema es un hub que facilita el intercambio de datos entre diferentes organizaciones o personas.

Mover datos médicos desde la base de datos de una organización a una persona u otra organización distinta es complicado por diferentes razones.

La primera razón es que se trata de un proceso manual. No existe ningún gran hub que permita a diferentes proveedores de salud intercambiar datos de forma rápida y eficiente. Por lo tanto, todo va mucho más lento.

La segunda razón es la complejidad de los datos médicos. Cada sistema utiliza su propia nomenclatura, se miden cosas en diferentes unidades, hay muchos campos de texto. Esto complica la comprensión de los datos.

Un sistema de gestión de datos médicos como RedMedic promueve un estándar único con el que todos se pueden comunicar. Además las transacciones dentro son automáticas. Ya no hay política entre medio, solo ordenadores que son mucho más eficientes.

En conclusión se podría decir que un sistema de gestión de datos médicos multiinstitucional crea una capa de comunicación para las diferentes organizaciones que en muchas ocasiones se tienen que hablar entre sí.

No obstante, este sistema no es solo un avance para los proveedores de salud, sino también para los muchos investigadores que necesitan muchos datos para desarrollar nuevos fármacos y tratamientos y analizar la eficacia de medicamentos que hay en el mercado.

El problema como he comentado antes es que no existe ninguna gran autopista por donde circulen todos los datos, con un mismo formato. RedMedic es aparte de un sistema de gestión, también una gran piscina de datos estandarizados. Es una fuente de información para investigadores por ejemplo.

A día de hoy, los centros de investigación compran directamente a hospitales información anonomizada sobre sus pacientes. El hecho de que esta información esté anonimizada limita su potencial ya que quedan huecos en muchas historias por rellenar, y estos datos perdidos no se pueden recuperar.

Interaktion mit dem System

Der Patient interagiert mit RedMedic über eine Desktop-Anwendung. Mit dieser App hat man die folgende Funktionen:

1) Sie können Ihre Gesundheitsdaten einsehen, sortiert nach Kategorien, Einfügungsdatum und nach dem Anbieter gefiltert. Ressource kommt von.

2) Sie können die Zugriffsberechtigungen auf Ihre Daten regeln, indem Sie die Kategorien ändern, auf die eine bestimmte Adresse zugreifen kann.

3) Konsultieren Sie die Krankengeschichte oder eine Kategorie davon eines anderen Benutzers im Netzwerk, auf den Sie Zugriff haben.

4) Verwalten Sie verschiedene Profile mit unterschiedlichen digitalen Signaturen und Adressen, die Ethereum zugeordnet sind. Eine Mutter kann zum Beispiel nicht nur ihr Konto, sondern auch das ihres Kindes über dieselbe Anwendung verwalten, indem Sie dieses Profil auf der Registerkarte Profile aktivieren.

5) Erzeugen Sie einen privaten Schlüssel, mit dem ein neuer Benutzer im Netzwerk registriert wird.
